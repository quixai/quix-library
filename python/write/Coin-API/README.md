# Coin API 
This sample will allow you to connect to Coin API to collect data.

## You will need

- A Quix account - [Signup here](https://quix.ai/signup)
- A Coin API key
- A Topic

## Parameters

Ensure you have changed '{placeholder:COIN_API_KEY}' to your Coin API key
