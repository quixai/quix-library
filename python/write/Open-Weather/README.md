# Open Weather API
This sample will allow you to connect to the Open Weather API

## You will need

- A Quix account - [Signup here](https://quix.ai/signup)
- Open Weather API account from [here](https://openweathermap.org/api) 

## Parameters

 - Open Weather API Key
